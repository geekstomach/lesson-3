Spring Boot приложение для хранения небольших файлов

  1. Регистрация новых пользователей
  2. Аутентификация
  3. Заливка файлов на сервер (файлы храняться в папке пользователя, пользователь видит только свои файлы)
  4. На главной странице одним списком отображаются все залитые файлы (их название, размер, дата заливки), и кнопки для удаления и скачивания файла
